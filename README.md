# Discord App

## Quick Start

### Собрать контейнеры приложения
```bash
$ make build
```

### Запустить приложение
```bash
$ make up
```

### Остановить приложение
```bash
$ make down
```

### Перезапустить приложение
```bash
$ make restart
```

### Swagger UI
1. `http://localhost:8801/swaggerui/#` - список всех методов приложения
